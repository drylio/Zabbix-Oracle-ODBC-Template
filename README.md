# **Zabbix-Oracle-ODBC-Template**

    {$ORACLE_ARCHIVELOGSIZE}
    {$ORACLE_ODBC_DSN}
    {$ORACLE_ODBC_PASS}
    {$ORACLE_ODBC_USER}
